# WhyMap

High performance minecraft world map with web-based interface and infinite zoom.
Your map will be presented at http://localhost:7542

## Setup

First install yarn packages for frontend:

```bash
cd src-vue
yarn install
cd ..
```

To run the mod:

```bash
gradle runClient
```

To compile `jar` file:

```bash
gradle build
```
